# ⬡ Data Management Orbi

Hub centralizado para relatórios Power BI com controle de acesso.

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```
Roda na porta 3001.

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Roda na porta 5173.

## Tecnologias
- **Frontend:** React + TypeScript + Tailwind CSS + Vite
- **Backend:** Node.js + Express + TypeScript
- **Banco:** SQLite (migrável para SQL Server)
- **Auth:** JWT + bcrypt
